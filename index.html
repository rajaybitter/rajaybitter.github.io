WHAT IS NEAT TETRIS?

Our project is an agent which has been trained to play the game of Tetris. This agent will operate within a neural network that is constantly being evolved by a genetic algorithm. This method of using a genetic algorithm to evolve Nueral Networks is popularly known as NEAT!
